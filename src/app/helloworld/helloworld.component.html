Hello world!

I am writing this guide as an angular app. The idea is that you clone the repo and look at the source code as you read through.

You'll need to have the angular cli installed!

Everything is component based, and so, although it's not the entry point of the app, to see the related code for the first examples, you need to navigate to src/app/helloworld. This text is in the html file. There's also the .ts file, which has (almost) all code, a styles file, and a spec file, which is for testing.

If you want to do someting more interesting than a static page, you want dynamic content, so let's start with a variable.
